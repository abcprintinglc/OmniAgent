OmniPrint Agents
================

Version: 0.1.0
Type: WordPress plugin scaffold

Purpose:
A scaffold for a WordPress-based print ordering, approvals, pricing, preflight, invoicing, and workflow platform for ABC Printing.

Included:
- plugin bootstrap file
- autoloader
- activation/deactivation hooks
- custom role setup
- starter database migration class
- admin menu scaffold
- module placeholders for products, organizations, orders, approvals, pricing, preflight, AI, integrations, and dashboard
- starter admin CSS/JS

Next recommended implementation order:

2. Add product CRUD admin screens
3. Add organizations and members CRUD
4. Add order entry and order detail records
